# Chat，一个基于 socket 的即时通讯 app(demo)  
  
![image](https://github.com/coffeehu/Chat/blob/master/README-Img/2-chat.png)  
  

##一、服务器端：
Linux + Tomcat + MySQL ，
Java 实现： socket 用于消息通讯；Servlet 用于登陆、获取列表、关注等其他操作。
